## Como publicar no GitHub Pages (passo a passo)

1. Crie um repositório novo no GitHub:
   - Nome sugerido: `portfolio` ou `portfolio-leandro`
   - Deixe como público.

2. Faça upload dos arquivos:
   - Na página do repositório, clique em **Add file → Upload files** e envie `index.html`, `style.css`, `script.js`, `README.md`.
   - Commit as alterações.

3. Ative o GitHub Pages:
   - Vá em **Settings → Pages**.
   - Em **Source**, escolha a branch `main` (ou `master`) e a pasta `/ (root)`.
   - Clique em **Save**. Em alguns segundos/minutos o site estará disponível em `https://<seu-usuario>.github.io/<nome-do-repo>/`.

4. Dicas rápidas:
   - Para alterar o e-mail mostrado, edite `index.html` (linha com <a href="mailto:...">) antes de subir.
   - Para trocar links de projetos, atualize as URLs dos cards no HTML.
   - Se preferir trabalhar localmente, instale o Git, clone o repositório, substitua os arquivos e faça `git push`.
